## Problem

All table captions are currently written as bold paragraphs above the table:

```
**Table N:** Caption text here.

| Col1 | Col2 |
|------|------|
| data | data |
```

LaTeX treats the bold caption as a regular paragraph, separate from the longtable environment generated by the pipe table. Page breaks can (and do) occur between the caption and the table body.

## Fix

Convert each caption to Pandoc's native table caption syntax. Place the caption on the line immediately **after** the table, prefixed with `: `. Delete the bold caption paragraph above. Pandoc will render this as a `\caption{}` inside the longtable, binding them together.

Target format:

```
| Col1 | Col2 |
|------|------|
| data | data |

: Table N: Caption text here.
```

Note: The `: ` prefix is Pandoc syntax. The caption text itself retains the "Table N:" label for numbering consistency.
